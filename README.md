# Student Management System – REST API with API Key Authentication

## 📌 Overview
This project is a Java-based REST API that performs CRUD (Create, Read, Update, Delete) operations for managing student records. The API is secured using API Key Authentication.

## ✨ Features
- Create Student Records
- Read Student Records
- Update Student Records
- Delete Student Records
- API Key Authentication
- RESTful API

## 🛠️ Technologies Used
- Java
- HttpServer
- HTML
- CSS
- JavaScript

## 📁 Project Structure
```
JavaCRUDServer.java
PythonCRUDClient.py
index.html
style.css
script.js
```

## ▶️ How to Run
1. Compile the Java source file.
2. Run the Java server.
3. Open `index.html` in your browser.
4. Use the application to perform CRUD operations.
